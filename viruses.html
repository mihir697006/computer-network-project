Viruses, Worms, and Trojans

Virus

A virus is a malicious program that attaches itself to legitimate programs or files. It spreads when the infected file or program is run, potentially damaging system files or slowing down performance.

Types of Viruses

  • Boot Sector Virus: Targets the boot sector and loads on system startup.
  • Direct Action Virus: Executes and infects specific files when a condition is met.
  • Resident Virus: Installs itself in memory and continues working without user interaction.
  • Multipartite Virus: Attacks multiple parts of the system simultaneously.
  • Overwrite Virus: Destroys original file content by overwriting it.
  • Polymorphic Virus: Changes its code to evade antivirus software.
  • File Infector Virus: Infects executable files and spreads when opened.
  • Spacefiller Virus: Hides in unused portions of a file, making detection difficult.
  • Macro Virus: Written in macro language, spreads through documents and spreadsheets.

Worm

Worms are standalone malware that spread through networks and systems without needing a host program or user interaction.

Types of Worms

  • Email Worm: Spreads through infected email attachments or links.
  • Internet Worm: Propagates via internet connections and exploits vulnerabilities in web servers or operating systems.
  • IM Worm: Spreads through instant messaging platforms by sending malicious links or files.
  • File-Sharing Worm: Disguises as popular files and spreads via peer-to-peer networks.
  • Network Worm: Exploits network security flaws to propagate across devices.

Trojan

Trojans appear as legitimate programs but perform malicious actions once activated. Unlike viruses or worms, Trojans do not replicate themselves.

Types of Trojans

  • Backdoor Trojan: Opens a backdoor for remote control of the system.
  • Downloader Trojan: Downloads and installs other malicious software.
  • Banking Trojan: Steals financial information like login credentials and credit card numbers.
  • Rootkit Trojan: Hides malicious activities and files from the user and security software.
  • Ransom Trojan: Encrypts user files and demands payment to restore access.
